# Simple Manga Viewer

## Description

Simple manga viewer demo for test [tauri](https://v2.tauri.app/) + react


### How to run

* WEB

        npm run dev

* Linux

        npm run tauri dev

  In some distros is necessary to set these variables

      WEBKIT_DISABLE_DMABUF_RENDERER=1
      WEBKIT_DISABLE_COMPOSITING_MODE=1
